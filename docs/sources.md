# Sources & References

This guide is an independent walkthrough compiled from Thales' publicly available
documentation and related community resources. Always confirm exact steps,
commands, ports, and sizing against the official Thales documentation for the
**specific CipherTrust Manager version** you are deploying, as details change
between releases.
